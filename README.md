# vue-pomodoro-timer

 A timer, with the technique applied of pomodoro clock. Demo: https://pomodoro-timer-996c1.firebaseapp.com/

## RUN

- Cloning project:

      git clone https://github.com/profalves/vue-pomodoro-timer.git

- Enter folder project by cmd and run:

      yarn

  or:

      npm i
- For development:

      yarn dev

  or:

      npm run dev

- For production:

      yarn build

  or:

      npm run build
